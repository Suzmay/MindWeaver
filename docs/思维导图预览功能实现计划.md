# MindWeaver 思维导图预览功能实现计划

## 1. 功能概述

为 MindWeaver 思维导图添加预览模式，使用户可以在非编辑状态下查看思维导图，具有以下核心特性：

- 无编辑按钮，专注于内容展示
- 可控制的连接线和节点波动动画效果
- 双击查看节点详细内容（无编辑功能）
- 保存和分享功能入口
- 可选的网格隐藏功能
- 自定义画布底色（基于节点颜色库）
- 编辑模式和预览模式的快速切换

### 3.2 连接线和节点波动效果动画

**技术原理**：
- 使用 Canvas 2D API 和 requestAnimationFrame 实现流畅动画
- 利用正弦和余弦函数生成波动效果
- 为连接线添加随机波动参数
- 为节点添加轻微的上下波动效果

**实现位置**：`CanvasRenderer.tsx`

**新增状态和引用**：
```typescript
const animationTimeRef = useRef(0);
const animationFrameIdRef = useRef<number | null>(null);
const [animationPhase, setAnimationPhase] = useState(0);
```

**波动效果实现**：
1. **连接线波动**：
   - 基于时间偏移改变贝塞尔曲线控制点
   - 使用 `Math.sin(animationTime + offset)` 生成波动
   - 波动幅度控制在 5-10 像素范围内

2. **节点波动**：
   - 轻微的上下浮动（±3像素）
   - 每个节点有不同的相位偏移
   - 使用 `Math.cos(animationTime + nodeId.hashCode())`

3. **动画循环**：
   ```typescript
   const animate = () => {
     animationTimeRef.current += 0.05;
     setAnimationPhase(animationTimeRef.current);
     drawWithCanvas2D();
     animationFrameIdRef.current = requestAnimationFrame(animate);
   };
   ```

## 7. 技术要点

### 7.1 性能优化

- 使用 `useRef` 存储动画时间，避免不必要的重渲染
- 动画只在预览模式且开启动画时运行
- 使用 `requestAnimationFrame` 确保流畅性
- 波动幅度控制在合理范围内

### 7.2 状态管理

- 预览模式状态作为顶层状态，向下传递给子组件
- 使用 React.memo 优化子组件重渲染
- 合理的依赖数组，避免过度重渲染

### 7.3 可访问性

- 按钮添加适当的 aria-label
- 确保键盘导航支持
- 提供足够的视觉反馈
